2 IPTables

2.1 命令

iptables  
iptables-save 查看当前 iptables 设置

2.1.1 iptables

iptables <options>
    -I, --insert <chain> <rule_num>
    -i, --in-interface <interface>
    -p, --protocol <protocol>
    -m, --match <match>
    -j, --jump <target>

iptables -I INPUT -i eth1 -p tcp -m tcp --dport 80 -j ACCEPT
